Where Do Responses Go?

All responses are saved to the CaseHub database:

  • intake_responses - Contains all form submissions
  • intake_items - Tracks status of each form (pending/in_progress/submitted)
  • intake_packages - Package status and client info

Staff can view responses in CaseHub at:
https://casehub.app/casehub → Questionnaires → Responses

Security Features
  • Token-based access - Each client has a unique URL
  • Passphrase protection - Additional password required
  • HTTPS encryption - All data encrypted in transit
  • Session cookies - Passphrase verified once per session
  • IP logging - All submissions tracked
