Como funciona

Selecione o tipo de peça, preencha os campos do formulário e clique em Gerar. O sistema monta a peça usando o modelo processual adequado. {% if ollama_online %} Ative a opção "Gerar com IA" para que o Maestro revise e melhore a redação jurídica automaticamente. {% else %} Quando o Ollama estiver rodando, você poderá ativar a opção "Gerar com IA" para revisão automática da redação. {% endif %} Após gerar, você pode copiar o texto ou baixar em formato DOCX/TXT.

Dica: Você também pode gerar peças diretamente a partir de um processo. Acesse o processo e clique em "Gerar Peça" para pre-preencher os campos automaticamente. Ou adicione ?process_id=ID a URL de qualquer peça.
