Ative pagamentos via Stripe

Para receber pagamentos online dos clientes, configure a integração Stripe:

  1. Crie uma conta em stripe.com
  2. No painel Stripe, obtenha as chaves de API (modo test ou live)
  3. Defina as variáveis de ambiente no servidor: 
    STRIPE_SECRET_KEY=sk_live_xxxxx
STRIPE_PUBLISHABLE_KEY=pk_live_xxxxx
STRIPE_WEBHOOK_SECRET=whsec_xxxxx
  4. Reinicie o {{ org_name|default('CaseHub') }}

Dica: Para testes, use chaves sk_test_ e pk_test_.
